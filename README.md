# Sample-projects

Assignments from UIB.

## bank_simulation.py
This Python program is a simple bank account simulator for a class assignment. You can check your balance, make deposits or withdrawals, apply interest, and see your recent transactions. If your balance goes over one million you get a bonus interest rate.

## card_game.py
This Python program is a console-based version of Wish Solitaire, a simple card-matching game made for a class assignment. Players pick cards from eight piles and try to match pairs with the same value to clear the deck. The game also lets you save and load your progress, see the current game state, and restart from the main menu.

## class_grade.py
This Python program is a simple grade management sytem made for a class assignment. You can keep track of your university courses and grades, list courses by subject or level, add new courses, update grades, and calculate your overall average. It also lets you save all your course and grade data to a file so you can pick up where you left off.

## music_app.py
This Python program models a simple music library using classes for artists, songs, and albums, made for a class assignment. Users can view album details, including track lists and artist information, and filter albums by artist name. The project demonstrates basic object-oriented programming concepts such as class relationships, methods, and instance tracking.
